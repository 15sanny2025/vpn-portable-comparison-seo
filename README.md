## FAQ

**1) What does “portable” mean here?**  
It means you can run it with minimal install friction and move between machines without babysitting setup.

**2) Does this comparison require an account?**  
Some VPNs do. The comparison calls out account friction versus quick-start portability.

**3) Is a Kill Switch actually necessary?**  
Yes if you care about leaks. If the tunnel drops, a **Kill Switch** is the difference between “safe” and “exposed.”

**4) What should I check first when testing speed?**  
Baseline your connection without VPN, then test nearby **Global Servers** first. Distance and congestion matter more than slogans.
